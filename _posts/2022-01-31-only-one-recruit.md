<div class = "card">
    <h2 style="text-align: center;">About Only One Recruit</h2>
    <p style="text-align: center;">Only One Recruit is a puzzle game designed around the theme 'Only One'  made for <a href="https://itch.io/jam/gmtk-2019" target="_blank" style="padding: 2px">GMTK Game Jam</a> <br /> The game was made with a team of 2, including me! <br /> A dungeon crawler game where each character has only one ability. They are able to use their ability only once per level. Each character and enemy has only one health. You can build up a party but can control only one character at time. </p>
</div>
